FIND THE PATH
Effect: Feature Action: Standard • Range: Personal • Duration: Sustained • Cost: 1 point per rank
You can find the path to anywhere well known to you. If you have the wrong locational information then the path will lead you where you think it is. When you gain this power choose a sense for the path to be sensed by. When activating choose a well known location and a path will form from where you are now to your destination. Only you can sense the path. The path does its best to form such that it is the shortest path that is possible for you to take (although not necessarily the safest). If there is no possible path then the path will take you as close as possible and then indicate a dead end. The path updates if your knowledge of where your destination is changes (correctly or otherwise), if the shortest possible path changes, or if you tell it to change (such as to take a detour or avoid an obstacle). You know how long the path is but might not be able to sense every detail of it.

EXTRAS

Entire Path: You know everything about the path such as how much of it requires Flight (assuming you have Flight) and every location it passes through. Flat +2 points.
Linked Attain Knowledge: This only applies if the Attain Knowledge base is between 2-4 points per rank for determining location. It might be more useful to simply take these powers separately but if linked you only need a single action to use both. see Attain Knowledge.
