Usage Notes

This page assumes you know all of the rules.
If you don't know what Transcendence is then leave it at 0 and ignore it.
There is no option for Transcendent equipment since God-like powers don't exist in mundane common items that can be easily obtained.
Transcendence and power level are automatically set to the lowest possible (with a Transcendence minimum of 0) so that if you want to use a Transcendent trait you must change Transcendence immediately beforehand.
Devices are created in the power section using removable.
The Image Path input box can be absolute or relative (to the file). Only this can be saved (for security reasons) and not any image given to the file upload button.
Ranks in the Equipment Advantage are also automatically set to the lowest possible whenever the equipment section is changed.
Some modifiers (such as those that change range, action, and duration) are automatically set and can't be set manually. Change the drop downs for range etc instead.s
In order to lack an ability put "--" in the ability input.
Redundant powers, equipment, advantages and skills (those with the same type and subtype) are automatically removed.
To buy equipment that has no impact on any other section you may use Feature.
This counter doesn't have a direct option for applying modifiers to existing traits, such as Impervious Toughness, you may use Feature of the same cost to represent these (fill in the text).
You must spend points in whole numbers, if you have an odd number of total ranks in skills you will see a half point.
The text area below is for loading and saving data since locally writing to a file requires other tools.
The upload box is checked before the text area when loading, to clear it use the "Clear Boxes" button provided or click on the upload button then hit escape or cancel. This is also true for the image.
Some modifiers will show ranks to represent cost so keep in mind that the number of applications of some modifiers is listed wrong with correct cost. Such as Increased/ Reduced Range going to/ from Perception range.
There is no easy way to have enhanced trait effect power level so instead apply the traits where needed and make a note of how many are enhanced.
This currently can't track how much damage effects deal which is therefore not considered for power level. Thus there are some cases were the power level is slightly off.
